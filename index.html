Model Proposal for Phishing Detection in Text Messages using Machine Learning

Machine Learning

El proyecto aborda la amenaza creciente del phishing por mensajes de texto mediante el uso de aprendizaje automático para su detección y prevención. Propone una solución que incluye la recopilación de datos, preprocesamiento, extracción de características con CountVectorizer y Tf-Idf, y clasificación mediante diversos algoritmos de aprendizaje automático. Entre los modelos evaluados, el Clasificador de Bosques Aleatorios destacó con una precisión del 97.94 %, demostrando su efectividad para diferenciar mensajes legítimos de los potencialmente maliciosos. Esta solución subraya la importancia de las técnicas de aprendizaje automático para reforzar la ciberseguridad frente a este tipo de ataques.

Classification and Detection of Knives and Guns Using Computer Vision

Machine Learning - Computer vision

El estudio propone un sistema basado en visión por computadora para detectar armas en videos de vigilancia, con el objetivo de mejorar la seguridad pública en áreas de alto riesgo como Lima, Perú. Utilizando conjuntos de datos etiquetados, técnicas de preprocesamiento y modelos avanzados como CNN, VGG16 y algoritmos de aprendizaje automático (Random Forest, AdaBoost, entre otros), el sistema demostró alta precisión, logrando una exactitud del 98.02% con VGG16, reduciendo falsos positivos y permitiendo alertas rápidas y confiables. Se plantea implementar esta tecnología en aeropuertos, escuelas y eventos públicos, y se sugieren futuras investigaciones para optimizarla en dispositivos de bajo costo y ampliar su capacidad para detectar más tipos de amenazas.

Predictive Modeling of COVID-19 Spread: A Big Data-Based Approach to Pandemic Management

LSTM - Flask - Deep Learning

El artículo desarrolla un marco predictivo para modelar y pronosticar la propagación de enfermedades infecciosas como el COVID-19, integrando diversas fuentes de datos, incluidas estadísticas de la OMS, indicadores socioeconómicos y datos de vuelos globales. La solución se basa en el uso de redes neuronales LSTM, capaces de procesar datos secuenciales para capturar dependencias a corto y largo plazo. La metodología incluye recopilación, limpieza y normalización de datos, seguido de un análisis exploratorio y modelado predictivo. Los resultados se despliegan en una plataforma web interactiva que permite consultas en tiempo real y escenarios personalizados, proporcionando una herramienta robusta para la toma de decisiones en la gestión de pandemias. Esta solución combina análisis estadístico y aprendizaje automático, ofreciendo predicciones precisas y apoyo estratégico frente a crisis de salud pública.

Intelligent job supply and demand search engine: a recommendation system based on characteristic vectors

KNN - Scraping - Flask

El proyecto presenta un sistema automatizado para conectar candidatos con ofertas laborales utilizando técnicas avanzadas de procesamiento de lenguaje natural (NLP), modelos preentrenados como Llama 3.1 y Word2Vec, y procesos de clasificación y agrupamiento. Se realiza scraping masivo de portales como Bumeran y perfiles de LinkedIn para recolectar datos sobre experiencia, habilidades y educación, almacenándolos en formatos estructurados como JSON y bases de datos PostgreSQL. Los datos son preprocesados y vectorizados para representar los perfiles y las ofertas en un espacio matemático. El sistema emplea algoritmos de recomendación basados en distancia euclidiana para comparar perfiles y ofertas, generando coincidencias personalizadas. Incluye interfaces web intuitivas para facilitar el uso por parte de candidatos y empleadores. Aunque el modelo actual alcanza un 60% de precisión en las recomendaciones, se propone mejorar su rendimiento mediante la integración de más características y refinamientos en el modelo.

Sistema de gestión de salas virtuales

.NET 8 - C#

He desarrollado un backend utilizando .NET 8 diseñado para gestionar y automatizar la administración de salas virtuales de Zoom. Este sistema permite registrar y centralizar las cuentas de Zoom, ofreciendo una plataforma eficiente para administrar múltiples usuarios y sus configuraciones. Además, el backend está integrado con un script en Python que interactúa directamente con la API de Zoom, habilitando la creación automática de salas virtuales según los parámetros definidos. Esta solución combina la robustez de .NET 8 para la gestión y procesamiento en el servidor con la flexibilidad de Python para la automatización, proporcionando una herramienta escalable y eficiente para simplificar la gestión de reuniones virtuales.
